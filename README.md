# Project-Web-Kelompok5-UAS
Repository untuk pengembangan proyek website kelompok 3. Setiap anggota mengerjakan fitur masing-masing menggunakan Git dan GitHub.
kami menggunakan ai sebagai berikut:
1. merapihkan desain agar tampilan website lebih rapi dan menarik
2. memperbaiki error untuk menemukan penyebab dan solusi dari error.
3. mendapatkan saran dari fitur yang akan dibuat.
4. Optimasi tampilan responsif agar website dapat menyesuaikan berbagai ukuran layar.
5. membantu mencari solusi jika kami menemukan kendala dalam proses pembuatan website.
